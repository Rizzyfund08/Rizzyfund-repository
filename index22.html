query Selector () vs getElementById


The apparent comparison between these twoconcepts is that they both select elements froma a web page. But they do so in various ways.


You may use a querySelector() statementstochoose an elements based on a Css Selector. This means you may select items based on their ID, class or any other form of a selector. You can only choose an elements by its ID when using the getElementById() function.

in general, You should choose the option that obviously does the task.
If you simply need to choose an element by it's ID or class , Use
getElementById or getElementByClassName.

NOTES, The querySelector() technique is the best choice if you need to apply
a more complex rule to choose items.

Both querySelector and getElementById have been around for a long in JavaScript.
Consequently, both of these approaches are entirely supported by current Browser's.
