WHAT ARE HEXADECIMAL DIGITS ?

hexadecimal refer to using base 16 asa numbering systems. what exactly does this mean?.
The most common numbering systems uses a base of 10. This means that for each digit, there are ten possible values:
  • Ten different values may be expressed with a single digit [0 - 9]
  • One hundred [ten times ten ] possible values may be expressed using two digits [0 - 99]
One number in a hexadecimal systems represent sixteen potential values. the fiorst ten numbers ae represent by 0 - 9, and the last six are A - F in hexadecimal rotation.

Because each hexadecimal digit may represent 16 different values, two digit can represent 16 * 16 for a total of 256.

CSS FONT STYLE

Font Selection Is Critical

Choosing the proper typeface significantly influence how reader interact with a websites.
The appropriate typeface may help your business establish a distinctive identity.
It is critical to have an easy-to-read typeface.The typeface improve the readability of your writing.It is also critical to choose a suitable font color and text size.

Generic Font Families

In css,there are Five generic font families:
  1. serif fonts -have a minor stroke at the edge of each letter. They create a sense of formality and elegance.
  2. Sans-Serif fonts- have clean lines (no small stroke attached). They create a modern and a minimalistics looks.
  3. Monospace fonts -here,all the letter have the same fixed width. They create a mechanical look.
  4. Cursive font- imitate human handwritting
  5. Fantasy fonts- are decorative/ playful fonts.
All the different font names belong to one of the generic font families.
To achieve optimal interoperability among browsers / operating sytems,the font families should store many font names as a "fallback" mechanism. Begin with the desired typeface and conclude with a generic family (to let the browser pick a similar font in the generic family if no other fonts are available). A comma should be used to seperate the font names.

TEXT ALIGNMENT + SIZING

TEXT ALIGNMENT

Text components are always aligned to the right of their container by default. use the TEXT-ALIGNED-ATTRIBUTES to adjust the alignment of text inside an element. there are four align atrributes:
  • CENTER: to center the text
  • LEFT: align the text to the left of it's container
  • RIGHT: align the text to the right of it's container
  • JUSTIFY: the text will pread out to fill out the full width of it's container

TEXT SIZING

The font-ize attribute allows you to modify the size of your text. The font-size prameter accept both relative and absolute values.The most often used absolute values is px, while the most commonly used relative values is ems, rems.
The em and rem are both relative measurement values that function similarly to percentages in that they act as a multiplier in relation to another units of measurements.
In Terms Of Size:
  • 1 em corresponds to the font-size of the element's parent.
  • 1 rem is the font font-size of the root element of the whole HTML page.
  • To change the height of a line of text, use the line height attributes.
